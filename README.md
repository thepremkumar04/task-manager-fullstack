# Task Manager Full-Stack Application

## 🚀 Overview
This is a full-stack task management application built using React and Django REST Framework with JWT authentication.

## 🛠 Tech Stack
- React.js
- Django
- Django REST Framework
- JWT Authentication
- Tailwind CSS
- Docker

## ✨ Features
- User authentication using JWT
- Create, update, delete tasks
- Responsive UI
- Secure REST APIs
- Dockerized deployment

## 📌 Future Improvements
- Task reminders
- Email notifications
- Role-based access
